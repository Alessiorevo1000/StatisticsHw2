 Queue:
        
A queue is a linear data structure that follows the First-In-First-Out (FIFO) principle. This means that the first item added to the queue is the first one to be removed.

       Enqueue, Dequeue:

        let value = myQueue.shift(); 

        



 Stack:
        
A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. This means that the last item added to the stack is the first one to be removed. 

       Push, Pop:
        myStack.push(value);  
        let value = myStack.pop();
